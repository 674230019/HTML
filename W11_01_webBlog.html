CATEGORY ARCHIVES: CSS DISPLAY PROPERTY

เปรียบเทียบ display: block vs inline-block

28 AUG

ตารางด้านล่างแสดงความแตกต่างระหว่าง display: block และ display: inline-block เพื่อช่วยให้เข้าใจการเลือกใช้ display property ที่เหมาะสม

คุณสมบัติ display: block display: inline-block
เรียงต่อกันในบรรทัดเดียว การจัดเรียงองค์ประกอบ 📦 แต่ละตัวอยู่บรรทัดใหม่ ✅
ตามเนื้อหาภายใน (auto) ความกว้างเริ่มต้น ❌ กินพื้นที่เต็มความกว้าง (100%) ✅
กำหนดได้ทั้ง width และ height การกำหนด width และ height ✅ กำหนดได้ทั้ง width และ height ✅
ใช้ได้ครบทุกทิศทาง margin และ padding ✅ ใช้ได้ครบทุกทิศทาง ✅
สามารถใช้ได้ (top, middle, bottom) vertical-align ❌ ไม่มีผลกับ block element ✅
มีช่องว่างจาก whitespace ใน HTML ช่องว่างระหว่างองค์ประกอบ 🚫 ไม่มีปัญหาช่องว่าง ❌
navigation menu, button groups การใช้งานที่เหมาะสม 📄 div, section, header, footer 🎯

สรุป: display: block เหมาะสำหรับโครงสร้างหลักของหน้าเว็บ ส่วน display: inline-block เหมาะสำหรับการสร้างเมนู หรือองค์ประกอบที่ต้องเรียงต่อกัน
